## Homepage

### Purpose
Introduce Surigao del Norte as a world-class destination and provide entry points to all major sections of the platform.

### Story
The homepage tells the story of Surigao del Norte as a place of natural beauty, cultural richness, innovation, and opportunity. It invites visitors to explore the province through tourism, investment, and engagement.

### Section Order

1. **Editorial Hero** — Cinematic introduction to Surigao del Norte
2. **Featured Destinations** — Highlight key tourism locations
3. **Cultural Showcase** — Celebrate local heritage and traditions
4. **Innovation Highlights** — Showcase progress and development
5. **Upcoming Events** — Promote current and future activities
6. **Quick Access** — Links to all major sections
7. **Newsletter CTA** — Encourage ongoing engagement

### Image Direction

**Hero:**
- Subject: Iconic Surigao del Norte landscape (Siargao, Cloud 9, Sugba Lagoon)
- Style: Cinematic, wide-angle, natural lighting
- Mood: Inspiring, welcoming
- Composition: Rule of thirds, leading lines

**Featured Destinations:**
- Subject: Specific locations (beaches, islands, landmarks)
- Style: Professional, high-quality
- Mood: Inviting, adventurous
- Composition: Varied for visual interest

**Cultural Showcase:**
- Subject: Festivals, traditions, daily life
- Style: Authentic, candid
- Mood: Warm, celebratory
- Composition: Human-centered

**Innovation Highlights:**
- Subject: Modern facilities, technology in use
- Style: Professional, clean
- Mood: Progressive, optimistic
- Composition: Balanced, contextual

### Content Strategy

**Hero:**
- Label: "WELCOME TO SURIGAO DEL NORTE"
- Heading: "Discover Paradise in the Philippines"
- Description: 40-50 words introducing the province
- CTA: "Explore Destinations →"

**Featured Destinations:**
- 3-4 destination cards
- Each with image, name, brief description
- Link to Activities page

**Cultural Showcase:**
- Editorial section with image and text
- Highlight specific cultural aspects
- Link to detailed content

**Innovation Highlights:**
- Grid of innovation stories
- Each with image, title, description
- Link to Framework page

**Upcoming Events:**
- List of 3-5 events
- Date, title, brief description
- Link to Activities page

### Animation Strategy

**Hero:**
- Image scale: 1.08 → 1.00 (1.2s)
- Text fade: Staggered upward
- Parallax: 20-30px on scroll

**Sections:**
- Reveal on scroll: Fade and slide
- Staggered for multiple elements
- Duration: 600-800ms
- Easing: Ease Out

**Cards:**
- Hover: Scale 1.02, shadow increase
- Duration: 300-400ms

### Scroll Behavior

- Smooth scrolling throughout
- Parallax on hero image
- Section reveals triggered at 10-20% viewport
- Consistent spacing between sections

### Desired User Emotion

**Initial:** Wonder and curiosity
**Middle:** Excitement and inspiration
**End:** Motivation to explore

## Implementation Notes

### Common Patterns

All pages follow these patterns:

1. **Editorial Hero** — Cinematic opening
2. **Content Sections** — Editorial storytelling
3. **Visual Evidence** — Photography throughout
4. **Clear CTAs** — Action-oriented
5. **Smooth Animations** — Subtle and refined
6. **Responsive Design** — Mobile-first

### Section Templates

Create reusable section templates:

- Editorial Section (image + text)
- Card Grid (uniform cards)
- Feature List (items with descriptions)
- Timeline (chronological content)
- Gallery (image collection)
- Form (input fields)

### Data Structure

Consistent data structure for all pages:

```typescript
interface PageData {
  hero: HeroData;
  sections: SectionData[];
  seo: SEOData;
}

interface HeroData {
  label: string;
  heading: string;
  description: string;
  cta: string;
  image: string;
}

interface SectionData {
  type: 'editorial' | 'grid' | 'list' | 'timeline';
  content: any;
  animation?: AnimationConfig;
}
```

## Best Practices

### DO

- Maintain consistent section order
- Use editorial storytelling throughout
- Prioritize photography and visual content
- Keep text minimal and impactful
- Ensure smooth animations
- Follow responsive design principles
- Maintain consistent spacing

### DO NOT

- Create marketing-style layouts
- Use startup-style compositions
- Overcrowd sections with content
- Use excessive animations
- Ignore mobile experience
- Sacrifice image quality
- Deviate from editorial approach

---

## Warnings

### Critical Considerations

1. **Never deviate from editorial storytelling** — This is the core approach
2. **Always maintain consistent patterns** — Predictability is key
3. **Never sacrifice mobile experience** — Most users access via mobile
4. **Avoid content overcrowding** — Less is more
5. **Never use generic imagery** — Only authentic photography

### Recommendations

- Regularly review page blueprints
- Gather user feedback on page flows
- Stay aligned with brand identity
- Maintain consistency across pages
- Optimize for performance and accessibility
