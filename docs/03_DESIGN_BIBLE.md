## Design Philosophy

PRIME SDN's design philosophy is rooted in editorial storytelling inspired by premium destination websites like Visit Singapore. The goal is to create an immersive, cinematic experience that introduces Surigao del Norte as a world-class destination.

### Core Philosophy

**"Photography tells the story. Design supports the narrative. Every element serves the experience."**

### Guiding Principles

1. **Photography-First** — Images dominate, text supports
2. **Editorial Composition** — Magazine-quality layouts
3. **Generous Spacing** — White space is a design element
4. **Minimal Text** — Few words, maximum impact
5. **Smooth Motion** — Subtle, refined animations
6. **Premium Quality** — World-class standards throughout

### Anti-Principles (What We Avoid)

- Marketing-style hero sections
- Startup-style compositions
- Government website aesthetics
- Dashboard-like interfaces
- Cluttered layouts
- Excessive animations
- Stock-looking imagery
- Sales-driven design

## Editorial Storytelling Principles

### Narrative Structure

Every page should tell a story with a clear beginning, middle, and end:

1. **Opening** — Cinematic hero that establishes emotional connection
2. **Development** — Sections that build understanding and engagement
3. **Climax** — Featured content that creates peak interest
4. **Resolution** — CTAs that inspire action

### Section Focus

Each section should have ONE primary message:

- **Hero:** Introduce the destination
- **Story:** Explain the context
- **Features:** Highlight specific aspects
- **Showcase:** Display examples
- **CTA:** Inspire action

Never crowd a section with multiple competing messages.

### Content Hierarchy

1. **Primary Message** — One main point per section
2. **Supporting Context** — Brief explanation
3. **Visual Evidence** — Photography that illustrates
4. **Action** — Clear next step (if applicable)

### Emotional Flow

Guide visitors through an emotional journey:

- **Wonder** → **Inspiration** → **Connection** → **Motivation**

Each section should advance this emotional progression.

## Photography Direction

### Image Quality Standards

All photography must meet these criteria:

- **Resolution:** Minimum 2000px width for desktop
- **Sharpness:** Crisp details, no blurring
- **Lighting:** Natural, professional lighting
- **Color:** Professional color grading
- **Composition:** Thoughtful, intentional framing
- **Authenticity:** Real photos from Surigao del Norte

### Preferred Subjects

#### Tourism & Landscapes
- Siargao coastline and beaches
- Cloud 9 surf breaks
- Sugba Lagoon
- Sohoton Cove
- Bucas Grande islands
- Mountain landscapes
- Ocean sunrises/sunsets
- Drone aerials
- White sand beaches
- Tropical forests

#### Culture & Community
- Local festivals and celebrations
- Traditional crafts and practices
- Community gatherings
- Daily life in barangays
- Cultural performances
- Local cuisine
- Traditional architecture

#### Innovation & Development
- Modern facilities and infrastructure
- Innovation hubs and co-working spaces
- Educational institutions
- Startup environments
- Technology in use
- Sustainable development projects

### Avoid These Subjects

- Generic conference/meeting photos
- Stock office imagery
- Crowded seminar photos
- Generic business settings
- Low-quality snapshots
- Overly saturated or edited images
- Watermarked or branded images

### Image Style

- **Cinematic:** Wide, sweeping compositions
- **Natural:** Authentic lighting and colors
- **Emotional:** Evokes feeling or aspiration
- **Professional:** High production quality
- **Contextual:** Shows Surigao del Norte specifically

## Image Composition Rules

### Hero Images

- **Full-width:** Span entire viewport width
- **Height:** 90-100vh for desktop
- **Overlay:** Subtle dark gradient for text readability
- **Subject:** Dominant, emotionally resonant
- **Composition:** Rule of thirds, leading lines

### Section Images

- **Size:** Large, occupying significant portion of section
- **Placement:** Alternating left/right for variety
- **Rounded Corners:** 28px radius for editorial feel
- **Shadows:** Soft, premium shadows (never harsh)
- **Aspect Ratio:** Landscape (16:9 or 3:2 preferred)

### Gallery Images

- **Grid:** Masonry or consistent grid layout
- **Spacing:** Generous gaps between images
- **Hover Effects:** Subtle scale (1.02-1.03)
- **Transitions:** Smooth, never abrupt
- **Loading:** Optimized with blur-up placeholders

### Portrait/Profile Images

- **Style:** Professional, authentic
- **Background:** Clean, contextual
- **Lighting:** Even, flattering
- **Expression:** Natural, engaging
- **Cropping:** Consistent across profiles

## Layout System

### Grid Structure

#### Desktop (1440px max)
- **Columns:** 12-column grid
- **Gutter:** 24px
- **Margins:** 80px left/right
- **Content Width:** 1280px max

#### Tablet (768px - 1024px)
- **Columns:** 8-column grid
- **Gutter:** 20px
- **Margins:** 60px left/right
- **Content Width:** 908px max

#### Mobile (< 768px)
- **Columns:** 4-column grid
- **Gutter:** 16px
- **Margins:** 24px left/right
- **Content Width:** 100% - 48px

### Section Layouts

#### Editorial Two-Column
- **Ratio:** 40% text / 60% image (or vice versa)
- **Alignment:** Content vertically centered
- **Spacing:** Generous gap between columns
- **Alternating:** Switch image position per section

#### Full-Width Hero
- **Structure:** Full-width background image
- **Content:** Overlaid on image
- **Alignment:** Left-aligned, max 600px width
- **Height:** 90-100vh

#### Three-Column Grid
- **Use:** Features, cards, showcases
- **Balance:** Equal or weighted columns
- **Responsive:** Stack on mobile
- **Spacing:** Consistent gaps

#### Masonry Gallery
- **Use:** Photo galleries, portfolios
- **Pattern:** Varied heights for visual interest
- **Spacing:** Consistent gaps vertically and horizontally
- **Loading:** Staggered reveal animation

## White Space Rules

### Vertical Spacing

#### Section-to-Section
- **Desktop:** 120-160px between sections
- **Tablet:** 80-120px between sections
- **Mobile:** 60-80px between sections

#### Element-to-Element
- **Headings to Body:** 24-32px
- **Body to CTA:** 32-40px
- **CTA to Next Element:** 40-48px
- **Image to Text:** 32-40px

### Horizontal Spacing

#### Margins
- **Desktop:** 80px left/right
- **Tablet:** 60px left/right
- **Mobile:** 24px left/right

#### Column Gaps
- **Desktop:** 24-32px
- **Tablet:** 20-24px
- **Mobile:** 16-20px

### Breathing Room

- **Never crowd content** — Always maintain generous spacing
- **Let images breathe** — Don't place text too close
- **Respect the viewport** — Don't fill every pixel
- **Create visual rhythm** — Consistent spacing patterns

## Responsive Philosophy

### Mobile-First Approach

Design for mobile first, then enhance for larger screens:

1. **Mobile Base** — Essential content and functionality
2. **Tablet Enhancement** — Additional columns, larger images
3. **Desktop Expansion** — Full layouts, maximum impact

### Breakpoints

- **Mobile:** < 768px
- **Tablet:** 768px - 1024px
- **Desktop:** > 1024px
- **Large Desktop:** > 1440px

### Responsive Behavior

#### Text Scaling
- **Headings:** Use clamp() for smooth scaling
- **Body:** 16px base, scale slightly on larger screens
- **Labels:** Maintain readability at all sizes

#### Image Sizing
- **Mobile:** Full width, appropriate height
- **Tablet:** Proportional scaling
- **Desktop:** Maximum impact, large format

#### Layout Changes
- **Mobile:** Single column, stacked
- **Tablet:** 2-column where appropriate
- **Desktop:** Full editorial layouts

### Touch Considerations

- **Tap Targets:** Minimum 44x44px
- **Spacing:** Generous gaps for touch accuracy
- **Gestures:** Support swipe for carousels/galleries
- **Feedback:** Visual feedback on touch

## Motion Principles

### Animation Philosophy

**"Motion should enhance, never distract. Subtle is always better than flashy."**

### Core Principles

1. **Purposeful** — Every animation has a reason
2. **Subtle** — Never overwhelming or distracting
3. **Smooth** — Fluid, never jerky or abrupt
4. **Fast** — Respect user's time (600-800ms max)
5. **Natural** — Follow physical expectations

### Animation Types

#### Reveal Animations
- **Fade Up:** Elements fade and move upward
- **Scale In:** Elements grow from smaller to full size
- **Slide In:** Elements slide from edges
- **Mask Reveal:** Content reveals through clipping mask

#### Scroll Animations
- **Parallax:** Background moves slower than foreground
- **Stagger:** Elements animate in sequence
- **Trigger:** Animations start when in viewport
- **Progress:** Animations tied to scroll position

#### Hover Animations
- **Scale:** Subtle growth (1.02-1.03)
- **Shadow:** Shadow intensity increases
- **Color:** Smooth color transitions
- **Transform:** Subtle position or rotation changes

### Timing

- **Fast:** 200-300ms (micro-interactions)
- **Standard:** 400-600ms (standard transitions)
- **Slow:** 600-800ms (hero reveals)
- **Never:** > 1000ms (too slow)

### Easing

- **Ease Out:** Decelerate (most common)
- **Ease In Out:** Accelerate then decelerate (smooth)
- **Linear:** Constant speed (rare, specific cases)
- **Custom:** Tailored for specific effects

---

## Scroll Behavior

### Parallax

#### Hero Parallax
- **Movement:** 20-30px vertical shift
- **Direction:** Opposite to scroll
- **Smoothness:** Very subtle
- **Performance:** GPU-accelerated

#### Section Parallax
- **Images:** Slight movement (10-20px)
- **Background:** Very subtle (5-10px)
- **Text:** Fixed (no parallax)
- **Consistency:** Same direction across page

### Smooth Scrolling

- **Duration:** 500-700ms for anchor links
- **Easing:** Ease In Out
- **Offset:** Account for fixed headers
- **Interruptible:** Allow user to interrupt

### Scroll Triggers

- **Threshold:** 10-20% of element in viewport
- **Once:** Animate only once per session
- **Delay:** Stagger for multiple elements
- **Performance:** Use Intersection Observer

## Interaction Philosophy

### Hover States

#### Buttons
- **Scale:** 1.02-1.03
- **Shadow:** Slight increase
- **Color:** Smooth transition
- **Duration:** 200-300ms

#### Images
- **Scale:** 1.02-1.03
- **Shadow:** Increase intensity
- **Brightness:** Slight brightening
- **Duration:** 300-400ms

#### Links
- **Underline:** Expand from center or left
- **Color:** Smooth transition
- **Arrow:** Slide slightly
- **Duration:** 200-300ms

### Click/Touch Feedback

- **Visual:** Immediate visual feedback
- **Scale:** Slight press effect (0.98)
- **Duration:** 100-150ms
- **Haptic:** (Mobile) Vibration if supported

### Focus States

- **Outline:** Clear, visible focus ring
- **Color:** Brand color or high contrast
- **Offset:** 2-3px from element
- **Accessibility:** Never remove

## Background Treatment

### Hero Backgrounds

- **Type:** Full-width image
- **Overlay:** Dark gradient (35-45% opacity)
- **Gradient:** Left to right fade
- **Purpose:** Text readability

### Section Backgrounds

- **Primary:** White (#FFFFFF)
- **Secondary:** Very light gray (#F8F9FA)
- **Accent:** Subtle blue tint (#F5F7FF)
- **Pattern:** Minimal, if any

### Avoid

- **Heavy gradients** — Keep subtle
- **Glowing blobs** — Not our aesthetic
- **Busy patterns** — Distracting
- **Dark backgrounds** — Except for hero overlays
- **Excessive decoration** — Minimal is better

## Visual Rhythm

### Creating Flow

#### Vertical Rhythm
- **Consistent spacing** between sections
- **Progressive disclosure** of information
- **Alternating layouts** for variety
- **Breathing room** between content blocks

#### Horizontal Rhythm
- **Consistent column widths** where possible
- **Balanced visual weight** across sections
- **Guided eye movement** through layout
- **Clear hierarchy** of elements

### Pacing

#### Fast Pacing
- **Hero sections** — Immediate impact
- **Feature grids** — Quick scanning
- **CTA sections** — Clear action

#### Slow Pacing
- **Story sections** — Reading time
- **Editorial content** — Absorption
- **Image showcases** — Appreciation

### Variety

- **Mix layouts** — Don't repeat same pattern
- **Vary spacing** — Create rhythm
- **Alternate image positions** — Keep interest
- **Change section heights** — Visual diversity

## UI Consistency Rules

### Typography Consistency

- **Font Family:** Montserrat (primary), Inter (secondary)
- **Headings:** Consistent sizing hierarchy
- **Body:** Consistent line height (1.6-1.8)
- **Colors:** Consistent text colors per context

### Color Consistency

- **Primary:** #3B82F6 (brand blue)
- **Secondary:** #0A1628 (dark navy)
- **Accent:** #60A5FA (light blue)
- **Neutral:** #64748B (gray text)
- **Background:** #FFFFFF (primary), #F8F9FA (secondary)

### Component Consistency

- **Buttons:** Same style across all pages
- **Cards:** Same border radius and shadow
- **Inputs:** Same styling and focus states
- **Modals:** Same behavior and appearance

### Spacing Consistency

- **Use spacing scale:** 4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80, 120, 160px
- **Section spacing:** Consistent across pages
- **Element spacing:** Follow hierarchy
- **Margins:** Consistent per breakpoint

### Animation Library

Create reusable animation variants:

```javascript
const fadeInUp = {
  initial: { opacity: 0, y: 40 },
  animate: { opacity: 1, y: 0 },
  transition: { duration: 0.6, ease: "easeOut" }
};

const scaleIn = {
  initial: { opacity: 0, scale: 1.05 },
  animate: { opacity: 1, scale: 1 },
  transition: { duration: 0.8, ease: "easeOut" }
};

const slideIn = {
  initial: { opacity: 0, x: 60 },
  animate: { opacity: 1, x: 0 },
  transition: { duration: 0.7, ease: "easeOut" }
};
```

## Best Practices

### DO

- Always prioritize photography and visual storytelling
- Maintain generous white space throughout
- Use refined, subtle animations
- Keep text minimal and impactful
- Ensure mobile-first responsive design
- Optimize for performance and accessibility
- Maintain consistency across all pages

### DO NOT

- Create marketing-style landing pages
- Use startup-style compositions
- Overcrowd sections with content
- Use excessive animations or effects
- Sacrifice image quality for performance
- Ignore accessibility requirements
- Deviate from editorial storytelling approach

## Warnings

### Critical Considerations

1. **Never deviate from editorial storytelling** — This is the core differentiator
2. **Always maintain premium quality standards** — Quality is the brand
